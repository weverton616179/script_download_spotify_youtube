# script_download_spotify_youtube
script simples em python para download de videos, musicas e playlists do spotify.

Requisitos
- pip install yt_dlp
- pip install spotdl
- baixar ffmpeg-git-essentials.7z no site https://www.gyan.dev/ffmpeg/builds/ extrair e adicionar ao PATH (variaveis de sistema)
- se cadastrar no https://developer.spotify.com/ criar uma aplicação e pegar o client_id e client_secret
